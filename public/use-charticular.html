Comparison with D3

I built the bubble chart with Charticulator. I think there are two major differences, the first one is the resource. When I try to build the chart by Charticulator, there are less templates I can use and I am not able to find my answers when I search for some issues. D3 has lots of template resource and tutorials, and I am able to find the solutions with my issues. The second one is the user interface, in my opinion, the user interface is not so straightforward, since it is a high level application to build a chart, I think it should make it easier to use.
